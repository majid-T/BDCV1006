"CSS"

CSS is short for Casscading Style Sheets and is used to add style to an html file. CSS have three kind of selectors where you can use tags by tagname in genereal Ex: H1, or you can define classes to use it in multiple HTML tags. Classes are defined by adding . in frong of your class name Ex: .myClassName. and you can also define an id using # and access specific HTML tag with that id in your page. Ex: #myId. Intresting fact about CSS selectors is that they can be chained to make a combinition and have more flexible selectors and styles.

What is an HTML tag? Give three examples of tags.

An HTML tag is used to define what is the element we are using in the page. They can be used for items we are displaying in the page or can hold information about meta data. Tags also can have attributes. Example of a meta data tag is link which can point to a style sheet or a java script file. Other tags can be mentioned are h a header tag which come in 6 sizes (H1,H2, ... ,H6) and p which defines a paragraph.

What is an empty tag? How does it differ from a regular tag?

An empty HTML tag is one that doesn’t contain anything. Examples are br and img tags which define a breack on line or a link to an image. Regular tags like p and h tags can hold text within them.

What is an attribute of an HTML tag?

An attribute defines more information about a tag. For example in <img> tag we have src and alt attributes where in src we define a location of the image either

What is the correct way for HTML tags to nest?

Every tag which is not an empty tag should be properly closed and we can add other tags in them in order. At the end we should have a tree like structure where tags are nested inside.

How do you add an id to an HTML tag?

id is an attribute for an html tag and is defined the same way every attribute is defined. we use <h1 id="myId">My Header</h1> to define an id and we can use it later for CSS or JS. This p tag for example has a sampleId define on it to be used in CSS for changing the color.

How do you add a class to an HTML tag?

Same as id, class is an attribute and can be defined by <p id="myClass"> My Paragraph </p>

By this approch we can give multiple elements same class and play with their style at the same time. It also save time reusing the style we defined.

Why would you use an ID vs a class?

Generally id is used when we seek a single element in the DOM where we use class attribute to define some p elements in red background while others in blue for example.

What is a type selector?

The CSS type selector matches elements by node name. It selects all elements of the given type within a document. Example is a element which can be selected by just defining a in the CSS file. After that all anchor elements if not defined otherwise will follow the style defined in the CSS file.

What is the head tag is for? What are some tags that go inside it?

<head> tag is used for defining metadata or sometimes to point at external resources such as javascript libraries we used in our document. It comes between <html> and <body>. Some tags used in them are <title>, <meta> and <script>

Body is the body tag is for? What are some tags that go inside it?

<body> tag represent the document body. Mainly most of the content of the page will be shown in this tag. Some tags used in them are <p>, <h1> , <a>, <img> and <table>
