"Client / server"

Client server is a type of architecture used it networks where there is a centeral server responding to requests made by clients for dynamic or static resources through URLs. Web servers in this architecture are responsible of generating the response for static resources where on dynamic pages this task is performed by application servers usally.

"HTML tags"

HTML tags are names user by HTMl to determine portions of the web page. It tells the brower how to display a page. We can have attributes in our html tags. Attributes gives more information . Attributes are always specified in the start tag. Attributes usually come in name/value pairs like: name="value". Last Version published is HTML5.

"Git"

GIT is a version control system developed by Linus Trevold. It helps you keep track of a projects and also makes it possible for multiple developers to work on a same project. Git can be used on your local machine but it can also have a remote reository, either on the company's servers where you work or on public sites which offer same services such as GitHub and Git Lab. More info and good tutorials can be found on GIT official web site.

"CSS"

CSS is short for Casscading Style Sheets and is used to add style to an html file. CSS have three kind of selectors where you can use tags by tagname in genereal Ex: H1, or you can define classes to use it in multiple HTML tags. Classes are defined by adding . in frong of your class name Ex: .myClassName. and you can also define an id using # and access specific HTML tag with that id in your page. Ex: #myId. Intresting fact about CSS selectors is that they can be chained to make a combinition and have more flexible selectors and styles.
W3 is officially managing CSS and last realesed version is CSS3. More info is available at W3 CSS homepage!
